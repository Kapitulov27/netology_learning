# netology.cloud_ansible

Кастомная коллекция Ansible для управления файлами.

## Модули

### my_own_file

Создает текстовые файлы на удалённом хосте с заданным содержимым.

| Параметр | Тип   | Обязательный | Описание       |
|----------|-------|--------------|----------------|
| path     | str   | да           | Путь к файлу   |
| content  | str   | нет          | Содержимое     |
| state    | str   | нет          | present/absent |

## Роли

### create_my_file

Role для создания файлов через модуль my_own_file.
